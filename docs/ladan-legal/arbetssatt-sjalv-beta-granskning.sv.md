# Arbetssätt — själva ta fram, jurist granskar

**Princip:** Ni skriver, samlar underlag och fyller i mallar i detta repo. **Ingen betald jurist** behövs för att *göra* jobbet. En jurist (t.ex. bekant eller fast arvode för genomgång) används **sparsamt** — för att dubbelkolla att paketet håller innan undertecknande och myndighetsärenden.

---

## Roller

| Roll | Vem | Gör |
|------|-----|-----|
| **Projektledning** | Kollega / familj | Action plan, tidslinje, samla fakta |
| **Stiftare / ägare** | Johan | Beslut, underskrift, donation |
| **Dokumentation** | Ert team | Markdown-utkast → PDF vid behov |
| **Jurist (granskning)** | Extern, ej utförare | 2–4 genomgångar (se nedan) |
| **Myndigheter** | Lantmäteriet, Länsstyrelsen, Skatteverket | Registrering enligt officiella blanketter |

---

## Granskningspass (när ni går till jurist)

Skicka **ett paket per pass** — inte löpande rådgivning.

| Pass | När | Innehåll till jurist | Fråga till jurist |
|------|-----|----------------------|-------------------|
| **G0** | Efter fas 0, innan Lantmäteriet | [`frågor-och-svar.sv.md`](frågor-och-svar.sv.md) (ifyllda **Svar** + sammanfattningstabell) + valt upplägg (en fastighet + arrende *eller* fastighetsbildning) | ”Är upplägget rimligt? Saknar vi något innan ansökan?” |
| **G1** | Färdiga utkast, före undertecknande | Stiftelseförordnande, arrende, gåvobrev, Johans avtal, bilaga avgränsning | ”Är formuleringarna giltiga och i rätt ordning?” |
| **G2** | Före lagfart / stiftelseregistrering | Signerade versioner + checklista myndighetsärenden | ”Något som stoppar registrering?” |
| **G3** | (Valfritt) Efter registrering | Kopior lagfart + registrerad stiftelse | ”Arkiv komplett?” |

**Budget jurist:** räkna med **fast pris per pass** (t.ex. 1–3 timmar granskning), inte timdebitering för att skriva avtal.

## Kvalitetskriterium innan G1

- [ ] Alla platshållare `[...]` i mallar är ifyllda eller medvetet tomma med kommentar
- [ ] Samma **fastighetsbeteckning** i alla dokument
- [ ] **Datum och parter** stämmer överallt
- [ ] **Ladan** är tydligt undantagen i gåvobrev och Johans avtal
- [ ] **Arrende** refererar till registrerad stiftelse (namn + org.nr när det finns)
- [ ] Barnen har **läst och godkänt** utkast (skriftligt i protokoll eller e-post sparad i arkiv)
- [ ] [`granskningspunkter-jurist.sv.md`](granskningspunkter-jurist.sv.md) ifylld

---

## Vad ni *inte* ska förvänta er göra själva

- **Bindande skatteråd** — använd Skatteverkets allmänna info; vid stora värden ev. förhandsbesked (kostar, men myndighet)
- **Fastighetsbestämning** — Lantmäteriet/lantmätare gör kartor
- **Tvist med granne** om servitut — kan kräja jurist om det eskalerar

---

## Ordning (oförändrad — viktigast)

1. Fakta och upplägg klart (fas 0) → **G0** valfritt men rekommenderat  
2. Utkast alla dokument (fas 1–5 i repo)  
3. **G1** → justera utkast  
4. Underteckna → myndigheter (fas 6)  
5. **G2** om jurist vill se signerade kopior före lagfart

Se [`action-plan.sv.md`](action-plan.sv.md) för detaljerade faser.
